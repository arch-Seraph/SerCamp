# SerCamp
A full-stack web application designed for camping enthusiasts to discover, share, and review campgrounds. Built as a comprehensive review platform, it allows users to create their own campground listings, upload images, and engage with a community of campers through a dynamic review system.

The project focuses on creating a seamless user experience while handling complex backend logic, including user authentication, database relationships, and responsive design.

## Key Features
Full CRUD Functionality: Create, Read, Update, and Delete campgrounds and reviews.

User Authentication: Secure login and registration system to ensure only authorized users can manage their posts.

Interactive Reviews: Leave ratings and comments on shared locations to help other travelers.

Responsive Design: A fully mobile-friendly interface built to look great on any device.

Dynamic Mapping: (Optional, if you added it) Integrated maps to visualize campground locations.

Frontend: HTML5, CSS3 (Bootstrap), EJS (Embedded JavaScript)
